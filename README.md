# SAT.Bahodir.uz

This is a minimal single-page prototype for an SAT study routine generator.

How to run

- Open `index.html` in your browser (double-click or use Live Server extension).

What it does

- Collects current score, target score, optional test date, focus areas, and daily available hours.
- Generates a 7-day sample routine and suggests cadence for full practice tests.
- Saves the profile in `localStorage` so you can load it later.
